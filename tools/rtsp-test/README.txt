NO uses estos scripts si ya tenés MediaMTX en ~/mediamtx-tapo.

Tu flujo (el que andaba):
  cd ~/mediamtx-tapo
  mediamtx mediamtx.yml

Luego en SimpleReplay (http://127.0.0.1:8080): Cámara IP → http://127.0.0.1:8888/tapo/

Los scripts start.sh / dev-local.sh de esta carpeta son solo prueba con video sintético
y pueden chocar con tus puertos 8554/8888/8889.
